## GPA的计算

厦马采用4.0绩点制（部分课程可能有差异）。

| 成绩区间 | 等级 | 绩点 |
|---------|------|------|
| 90-100 | A+ | 4.0 |
| 80-89 | A | 4.0 |
| 75-79 | A- | 3.7 |
| 70-74 | B+ | 3.3 |
| 65-69 | B | 3.0 |
| 60-64 | B- | 2.7 |
| 55-59 | C+ | 2.3 |
| 50-54 | C | 2.0 |
| 45-49 | C- | 1.7 |
| 40-44 | D | 1.0 |
| <40 | F | 0.0 |

## GE课程（General Education）

GE 课程是所有专业必修的通识课程，包含 MPU（马来西亚公民教育）、语言类、人文社科类等。

### 选课技巧

- 尽早选择口碑好的老师（通过学长学姐打听）
- GE 课程对GPA影响不小，不要掉以轻心
- 部分 GE 课程只在特定学期开设
