## 为什么继续留学

从厦马申请海外研究生项目是许多同学的选择。厦马的国际化背景和英语授课经历是申请的天然优势。

## 国家/地区的选择

| 目的地 | 特点 |
|--------|------|
| 英国 | 1年制硕士，性价比高，QS名校多 |
| 澳大利亚 | 环境好，移民友好，1.5-2年 |
| 新加坡 | 亚洲顶尖，NUS/NTU竞争激烈 |
| 美国 | 顶尖名校，竞争最激烈，奖学金机会多 |
| 德国 | 理工科强，部分公立免学费 |
| 日本 | 性价比高，文化亲近 |
| 香港 | 亚洲顶尖，申请相对容易 |
| 马来西亚本地 | 留在 UM/UTM 等本地高校读研 |

## 读什么学位/专业

- **转方向**：厦马的跨学科背景允许你在读研时转换方向（如本科经济转读数据科学硕士）
- **对口申请**：延续本科专业，深耕专业知识

## 润学

出于各种原因考虑移民/永居海外的同学，可关注"润学"相关经验贴。马来西亚本身对中国人有一定的移民通道（如 MM2H 计划）。

## 在厦马期间需要准备什么

### 外语考试

**英语**（雅思/托福/PTE）
- 英国院校多要求雅思 6.5，顶尖院校要求 7.0
- 建议大三前完成考试，为申请留出时间
- 厦马英语授课背景可写入文书，但仍需官方成绩

**德语**
- 申请德国公立大学通常需要 B2-C1 级别德语
- 部分英语授课项目无需德语

**日语**
- 部分日本大学要求 N2 以上
- 通过 JLPT 官方考试证明

**GRE**
- 申请美国理工科和部分商科项目需要
- 难度较高，建议提前6个月备考

### 科研经历

参见：[寻找科研和项目经验](../学习篇/科研与项目经验.md)

### 竞赛经历

**如何参与竞赛**：
- 关注学院和学校公告
- 加入相关学生社团
- 参加 ACM/ICPC（计算机）、数学建模、商赛等

### 推荐信

- 至少需要2-3封推荐信（多数为学术推荐）
- 提前维护与老师的关系，不要临时抱佛脚
- 给老师提供你的简历和想申请学校的信息

### 实习经历

实习经历对申请商科、新闻、传播等专业尤为重要。

## 警惕中介

留学申请市场中介良莠不齐，不少中介收费高昂且服务质量差。建议：
- 优先通过学长学姐获取经验
- 自己动手申请（DIY）成本低，且对申请过程更了解
- 若使用中介，选择口碑好、有实际案例的
