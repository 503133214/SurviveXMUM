## 三大账号

### 1. Campus ID — 最常用的那套

| 项目 | 说明 |
|------|------|
| **账号格式** | 你的 Campus ID（不是学号，形如 `XXX1234567`，卡片上印着） |
| **密码** | 初始一般为护照号后六位；改过就用你设的新密码 |
| **管哪些系统** | Moodle · AC Online · E-Services · ECard · AskA · AskA Maintenance · 校园 Wi-Fi |
| **重置密码** | https://id.xmu.edu.my/accounts/password/reset/?next=/accounts/update/ → 输 Campus ID → 查 **学校邮箱** 收重置链接 |

> 💡 这堆系统的密码是**共享同步**的——改了 Campus ID 密码，Moodle / AC / E-Services 等全跟着变。

---

### 2. CAS 账号 — Academic Portal 签到用

| 项目 | 说明 |
|------|------|
| **账号格式** | Campus ID（和上面同一个 ID，但走的是 CAS 认证通道） |
| **密码** | 初始一般也是护照号后六位；首次登录强制改 |
| **管哪些系统** | Academic Portal（签到 / 出勤率）· 厦大本部课程中心 `lnt.xmu.edu.cn` |
| **密码找回** | https://cas.xmu.edu.my/security/#/password/passwordFound → 填登录账号 + 身份证号/护照号 + 验证码 → 查校园邮箱收验证码 |

### ③ 学校邮箱 Outlook (Microsoft 365)

| 项目 | 说明 |
|------|------|
| **邮箱地址** | **`学号@xmu.edu.my`**[2](@ref) |
| **初始密码** | 同样是护照号后六位（IT 会在邮件里告知）[2](@ref) |
| **登录入口** | https://www.outlook.com → 用邮箱地址 + 密码登入 |
| **改密码** | 登入后点右上角齿轮 → 改密码；或 https://mysignins.microsoft.com/security-info/password/change |
| **忘了密码** | 带 E-Card / IC / 护照去 **图书馆流通台**，或邮件 `it@xmu.edu.my` 附证件照，或开 AskA 附证件照[2](@ref) |
| **⭐ 重要性** | 学校一切正式通知走这里 → **每周至少查一次**，别只守微信 |

> 毕业/离校后一段时间内邮箱会被停用清除，重要邮件自己提前存档。[2](@ref)

## 支付 / 账单相关

- **学费 & 住宿费** → **[XMUM ePayment Portal](https://epayment.xmu.edu.my/)** → 选 *Academic Related Payment* → 登录 ID 用 **本地 IC号(无`-`)** 或 **国际 护照号**[1,4](@ref)
- **ECard 充值 / 打印费 / 电费** → 同一 ePayment 门户 → 选 *Campus ECard Top Up* → 用 **Campus ID + Campus ID 密码** 登[1,4](@ref)

## 排查

1. **确认你登的是正确的账号体系** → 是不是把 CAS 密码套去 Moodle 了？（两套可能不同步）
2. **清浏览器Cookies / 用无痕模式 / 换浏览器** → 尤其Campus ECard Top Up这个页面，有时会抽风（FreePainter错误），需要清楚一下Cookies.
3. **VPN 场景** → 校外访问某些数据库用 https://webvpn.xmu.edu.cn/login（这是本部 VPN，不是校园网）
4. **以上都不行** → 询问 AskA（`app.xmu.edu.my/AskA`）或邮件 `it@xmu.edu.my`
